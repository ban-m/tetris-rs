## Short summary

関数型言語のコーディングとターミナルの入出力に慣れるためのテトリス

## How to play

```
git clone
cargo run --release
```

+ right: move right
+ left: move left
+ down: down
+ up: set immediately
+ x: reserve current block
+ c: right rotate
+ z: quite

左右がわからないので、右と左の動きが逆の可能性あり。
