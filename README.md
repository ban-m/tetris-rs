## Short summary

関数型言語のコーディングとターミナルの入出力に慣れるためのテトリス

## How to play

```
git clone
cargo run --release
```

+ d: move right
+ a: move left
+ s: down
+ w: set immediately
+ k: reserve current block
+ j: right rotate
+ q: quite
